## Features

- **Right-Click Generation**: Generate alt text for any image with a right-click
- **Multiple Modes**: Choose from Short, Long, or SEO modes
- **Quick Copy**: Copy generated alt text to clipboard instantly
- **Image Preview**: See the image while generating alt text
- **Offline Support**: Works with locally stored images

## Installation

### Chrome

1. Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/gistmag-alt-text-generato/fdlepmdamhcpnlnkpmpkgdhfelogkahi)
2. Click **Add to Chrome**
3. Click **Add Extension** to confirm

### Edge

1. Visit the [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/gistmag-alt-text-generato/piejclnjcjmlgneiecnnjgmnknlbpbje)
2. Click **Get**
3. Click **Add Extension** to confirm

### Firefox

1. Visit the [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/gistmag-alt-text-generator/)
2. Click **Add to Firefox**
3. Click **Add** to confirm

## Configuration

### 1. Get Your API Key

1. Sign up at [GistMag Dashboard](https://gistmag.co.uk/dashboard)
2. Navigate to **API Keys** section
3. Create a new API key
4. Copy your API key

### 2. Configure the Extension

1. Click the GistMag extension icon in your browser toolbar
2. Click **Settings**
3. Paste your API key
4. Choose default settings:
   - **Default Mode**: Short, Long, or SEO
   - **Default Language**: Language code
5. Click **Save**

## Usage

### Generate Alt Text

1. Right-click on any image on a webpage
2. Select **Generate Alt Text with GistMag**
3. Choose your mode (if not using default)
4. Wait for generation (usually 2-5 seconds)
5. Copy the generated alt text

### Quick Actions

- **Copy Alt Text**: Automatically copies to clipboard
- **View Details**: See all alt text variants (short, long, SEO)
- **Regenerate**: Generate again with different settings

## Mode Guide

### Short Mode
- **Best for**: Icons, thumbnails, simple images
- **Length**: 50-100 characters

### Long Mode
- **Best for**: Complex images, detailed scenes
- **Length**: 100-200 characters

### SEO Mode
- **Best for**: Blog images, product images, content images
- **Features**: SEO-optimized with keywords

## Troubleshooting

### Extension not working
- Verify your API key is correct
- Check your API usage limits
- Ensure you're connected to the internet
- Try refreshing the page

### Alt text not generating
- Check that the image URL is accessible
- Verify your API key is valid
- Check your credit balance
