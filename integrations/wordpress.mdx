## Overview

The GistMag WordPress plugin automatically generates AI-powered alt text for images in your WordPress media library, post editor, and featured images.

## Features

- **Automatic Generation**: Generate alt text when images are uploaded
- **Bulk Processing**: Process multiple images at once
- **Media Library Integration**: Works seamlessly with WordPress media library
- **Post Editor Support**: Generate alt text directly from the post editor
- **Featured Images**: Automatically handle featured images
- **Multiple Modes**: Choose from Short, Long, or SEO modes

## Installation

### From WordPress.org

1. Log in to your WordPress admin panel
2. Go to **Plugins** → **Add New**
3. Search for "GistMag Alt Text Generator"
4. Click **Install Now**
5. Click **Activate**

### Manual Installation

1. Download the plugin ZIP file
2. Go to **Plugins** → **Add New** → **Upload Plugin**
3. Choose the ZIP file and click **Install Now**
4. Click **Activate**

## Configuration

### 1. Get Your API Key

1. Sign up at [GistMag Dashboard](https://gistmag.co.uk/dashboard)
2. Navigate to **API Keys** section
3. Create a new API key
4. Copy your API key

### 2. Configure the Plugin

1. Go to **Settings** → **GistMag Alt Text**
2. Paste your API key
3. Choose default settings:
   - **Default Mode**: Short, Long, or SEO
   - **Default Language**: Language code (default: English)
   - **Auto-generate on upload**: Enable/disable automatic generation
4. Click **Save Settings**

## Usage

### Automatic Generation

When enabled, alt text is automatically generated when you upload images to the media library.

### Manual Generation

1. Go to **Media** → **Library**
2. Click on an image
3. Click **Generate Alt Text** button
4. Choose your mode (Short, Long, or SEO)
5. Click **Generate**

### Bulk Processing

1. Go to **Media** → **Library**
2. Select multiple images
3. Choose **Bulk Actions** → **Generate Alt Text**
4. Click **Apply**
5. Select mode and click **Generate**

### From Post Editor

1. Add an image to your post
2. Click on the image
3. In the image settings, click **Generate Alt Text**
4. The alt text will be automatically filled in

## Mode Guide

### Short Mode
- **Best for**: Icons, thumbnails, navigation images
- **Length**: 50-100 characters
- **Example**: "Golden retriever in park"

### Long Mode
- **Best for**: Hero images, complex scenes, infographics
- **Length**: 100-200 characters
- **Example**: "A happy golden retriever running through a lush green park on a sunny day"

### SEO Mode
- **Best for**: Product images, blog posts, landing pages
- **Features**: Includes relevant keywords
- **Example**: "Golden retriever dog playing fetch in park - pet photography"

## Troubleshooting

### Alt text not generating
- Verify your API key is correct
- Check your API usage limits
- Ensure images are publicly accessible

### Plugin not appearing
- Make sure the plugin is activated
- Check for plugin conflicts
- Verify WordPress version compatibility
